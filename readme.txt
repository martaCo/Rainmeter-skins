La skin è sostanzialmente uno snippet che lavora su framework Raindrop. Raindrop è un programma che introduce gadget, similmente a quelli visti con Windows 7, per il desktop.
Le skin auitano a tenere sotto controllo l'hardware ed il software; per esempio, è possibile controllare continuamente la quantità di disco utilizzata, la CPU o la/e RAM in uso. Dopo numerose ed attente ricerche su Internet, abbiamo visto che non ci sono skin per la misurazione della temperatura della CPU.
Perché è importante sapere quanto è calda la CPU? È forse uno dei fattori più importanti, il calore della CPU, poiché solo da quello si evince quanto il calcolo pesi su essa; oltre a ciò, è utile sapere quanto scalda, in modo da rendersi conto quando è opportuno aprire il PC per una pulizia ed un eventuale cambio di pasta termica.
Un normale utilizzo va dai 30 °C fino ai 80-85 °C; oltre i 90 °C la CPU è molto calda, e oltre i 95 °C si rischia un surriscaldamento che può portare all'arresto immediato del sistema.


COME INSTALLARE LA SKIN

1. Per beneficiare di questo piccolo software, è necessario scaricare, se non già presenti nel sistema, due software.
Scaricare ed installare Rainmeter.
Scaricare ed installare CoreTemp od un analogo software per l'interfaccia sistema-sensoristica.
Perché Rainmeter: esso è il framework necessario per il parsing delle istruzioni contenute nel file .ini, che rappresenta la skin vera e propria. Senza, il file non è chiaramente riconoscibile dal sistema privo di tale framework.
Perché CoreTemp: Rainmeter non è in grado di interfacciarsi direttamente coi sensori hardware della vostra macchina. CoreTemp serve per far sì che hardare e software si parlino.

2. Nella raccolta Documenti appare una cartella Rainmeter; si acceda a tale cartella.
Creare una cartella con un nome a piacere, quindi incollare il file .ini scaricato.
		
3. Andando sulla console di Rainmeter (basta un semplice tasto destro del mouse su una skin già presente sul desktop, quindi selezionare "Gestisci skin"), dopo qualche secondo appare, nella struttura ad albero delle cartelle predefinite, la nuova directory. Accedendovi, si trova il file .ini copiato, quindi si clicca sul pulsante "Carica".

NOTA
A questo punto la skin è caricata sul desktop e visibile. Nella versione attuale (1.0), la skin viene caricata in alto a sinistra (coordinate x=1, y=1).
Dato che è molto minimale, è facile non vederla. Per spostarla, basta fare drag-n-drop sul desktop, tracinandola (nella versione attuale (1.0) la skin è molto poco sensibile, quindi bisogna essere precisi nel selezionarla - a questo scopo si consiglia di puntare il cursore sulla scritta dei gradi).


NOTA
Senza CoreTemp, la skin segnerà sempre 0 gradi; pertanto, consigliamo vivamente di avviare CoreTemp con l'avvio di Windows.


NOTA
Questa skin è "intelligente", cioè è sensibile alla temperatura della CPU, e non è statica: questo significa che il cerchio attorno ai gradi si adatterà sulla base della temperatura attuale. Per esempio, se il processore sale a 89 °C, il cerchio si completa. Se rimane sopra gli 80 °C, il cerchio comincerà a restringersi, fornendo una grafica più utile di un cerchio pieno, che non mostra (graficamente) dei cambiamenti di temperatura.




The skin is constantly a snippet that works on Raindrop framework. Raindrop is a program that introduces gadgets, similar to those seen with Windows 7, for desktops.
The skins help to keep the hardware and software under control; for example, you can continuously monitor the amount of disk used, the CPU or RAM in
use. After numerous and careful searches on the Internet, we have seen that there are no skins for measuring the temperature of the CPU.
Why is it important to know how hot the CPU is? It is perhaps one of the most important factors, the heat of the CPU, since only from that it can be deduced how much the calculation of the weights on it; in addition to this, it is useful to know how much it heats up, in order to realize when it is appropriate to open the PC for a cleaning and a possible change of thermal paste. Normal use ranges from 30 ° C to 80-85 ° C; over 90 ° C the CPU is very hot, over 95 ° C there is the risk of overheating which can lead to the immediate stop of the System.


HOW TO INSTALL THE SKIN

1. To receive this small software, two software must be downloaded, if not already present in the system.
Download and use Rainmeter.
Download and use CoreTemp on a similar software for the sensor-plant.
Why Rainmeter: is the necessary framework for analyzing the instructions given in the .ini file, which describes the actual skin. Without it, the file is not clearly recognizable by the system without this framework.
Why CoreTemp: Rainmeter is not able to interface directly with the hardware sensors of your machine. CoreTemp serves to far away that hardare and software yes speak.

2. A Rainmeter folder appears in the Documents collection; access this folder.
Create a folder with any name, then paste the registered .ini file.

3. Going to the console of Rainmeter (just a simple button to destroy the mouse on a skin already on the desktop, then select "Manage skin"), after a few second, the new directory appears in the default folder tree. When you log in, you find the copied .ini file, then click on the button "Charge".


NOTE
At this point the skin is loaded on the desktop and visible. In the current version (1.0), the skin is loaded in the upper left corner (coordinate x = 1, y = 1).
Since it's very minimal, it's easy to miss. To move it, just click and drag on the desktop, tracing it (in the current version (1.0) the skin is a lot not very sensitive, so be precise in selecting it - for this purpose it is advisable to point the cursor at the writing of the degrees).


NOTE
Without CoreTemp, the skin will always mark 0 degrees; Therefore, it is strongly recommended to follow CoreTemp with the Windows update.


NOTE
This skin is "intelligent", that is sensitive to the temperature of the CPU, and it is not static: this means that the circle around the degrees will adapt on the base of the current temperature. For example, if the processor is sold at 89 ° C, the circle is complete. If it stays above 80 ° C, the circle will begin to shrink, provided a more useful graphic than a full circle, which does not show (graphically) temperature changes.
